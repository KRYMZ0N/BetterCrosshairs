### Requirements
* [MelonLoader](https://melonloader.co/) (v0.6.0+)
* [Schedule I](https://store.steampowered.com/)

### Installation
1. Download the latest [MelonLoader Installer](https://melonloader.co/download.html).
2. Run the installer and select your *Schedule I* game executable.
3. Ensure the **Il2Cpp** runtime is selected (as *Schedule I* runs on the Il2Cpp Unity backend).
4. Launch the game once to allow MelonLoader to generate the necessary `Mods` and `UserData` folders.
5. Download the latest `BetterCrosshairs.dll` from the [Releases](https://github.com/KRYMZ0N/BetterCrosshairs/releases) page.
6. Drag and drop the `.dll` file into your game's `/Mods` folder.
7. Restart the game and you're ready to go!
